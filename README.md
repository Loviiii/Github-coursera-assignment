# Simple Interest Calculator

## Description
This project contains a shell script to calculate simple interest.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## How it works
The script takes input from the user:
- Principal amount (P)
- Rate of interest (R)
- Time in years (T)

It then calculates simple interest using the formula above.

## Example
If:
P = 1000  
R = 5  
T = 2  

Then:
Simple Interest = (1000 × 5 × 2) / 100 = 100

## Usage
Run the shell script and enter the required values when prompted.
